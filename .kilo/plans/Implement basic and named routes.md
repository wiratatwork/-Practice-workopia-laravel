# Plan: Implement Basic and Named Routes in Laravel

The goal is to practice basic routing and named routes in the Laravel application `workopia` by adding three specific routes to `routes/web.php`.

## Tasks

1.  **Basic Route**: Create a GET route for `/contact` that returns `<h1>Contact Us</h1>`.
2.  **Named Route**: Create a GET route for `/all-jobs` named `jobs.index` that returns `Available Jobs`.
3.  **Route Helper**: Create a GET route for `/test-link` that returns an HTML anchor tag linking to `jobs.index` using the `route()` helper.

## Implementation Steps

1.  **Inspect `routes/web.php`**: Read the existing content of `routes/web.php` to ensure correct placement of the new routes.
2.  **Add Routes**:
    - Add `Route::get('/contact', ...)`
    - Add `Route::get('/all-jobs', ...)->name('jobs.index')`
    - Add `Route::get('/test-link', ...)` using `route('jobs.index')`.
3.  **Verification**: (Optional but recommended) Run a check to ensure the routes are correctly registered.

## Definition of Done
- `/contact` returns `<h1>Contact Us</h1>`.
- `/all-jobs` returns `Available Jobs` and is named `jobs.index`.
- `/test-link` returns a link to `/all-jobs` generated by the `route()` helper.
